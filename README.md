## Description

A starter project using [Nest](https://github.com/nestjs/nest) framework and [Firebase Functions](https://firebase.google.com/docs/functions).

## Installation

```bash
$ npm install
```

## Running the app

```bash
$ npm start

```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Deploy

```bash
$ npm run deploy

```
